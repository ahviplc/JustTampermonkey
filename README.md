## Install

`安装依赖`

```sh
npm i
npm install
或者
yarn
yarn install
```

## Usage

```sh
yarn vite
```

`使用 访问我`

```markdown
vite dist --port 8696

  vite v2.3.8 dev server running at:

  > Local: http://localhost:8696/
  > Network: use `--host` to expose
```
